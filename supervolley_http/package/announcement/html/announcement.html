中国大学生排球联赛(CUVA)吉祥物形象征集大赛

活动背景：

2018年，北京华羽时代体育文化发展有限公司牵手中国大学生体育协会独家运营中国大学生排球联赛，什么样的形象能让你一下子就联想起大学生排球呢？ 什么样的形象才能真正代表我们大学生排球人的特点呢？我们也一直在寻找，寻找一组（男票和女票哦）独特的小精灵，它或许聪慧可爱，或许朝气蓬勃，它将是CUVA的形象代表。 亲，无论你是涂鸦达人，还是怪才设计师，只要你够大胆，够有想象力，那就不要吝惜你的才华，拿起画笔或鼠标尽情挥洒你的想象力，把你心目中的那个可以代言CUVA的形象展示出来吧！ 让自己的作品成为CUVA的吉祥物吧！大学生排球联赛吉祥物形象征集大赛火爆开幕！

活动内容及目的：

征集符合大学生阳光、青春与活力兼具的比赛吉祥物形象，打造中国大学生排球联赛(CUVA)赛场形象的品牌。

活动时间：

2018年6月15日——2018年8月14日

作品要求：

1.吉祥物形象设计应围绕中国大学生排球联赛(CUVA)特点，紧紧围绕赛事包装的“M.V.P”（music volleyball party，排球音乐盛宴）的概念，有鲜明的象征意义；

2.吉祥物形象要求结合“CUVA”logo原图 （百度云下载地址链接: https://pan.baidu.com/s/1MreJ4ipvFzuO434EVL9wLA 密码: jfeh）进行创作，创意新颖、活泼可爱、色彩明快、构思巧妙、寓意深刻， 不仅与中国大学生排球联赛(CUVA）特色相结合，更要展现当代大学生积极向上，奋发有为的阳光青春的时代精神。

3.吉祥物形象应适用于网站、印刷品、服装、徽章等传播载体，易于识别记忆和推广宣传；

4.设计的吉祥物形象不得违反《中华人民共和国商标法》的有关规定，且吉祥物形象属原创并未被使用；

5.参赛作品注重打造中国大学生联赛整体IP形象为基准，吉祥物形象设计完整；

6.参赛作品提交时，需提交一组（男票和女票哦）的完整吉祥物图案并附有吉祥物名字哦。

提交细则：

1.吉祥物形象设计作品以电子版形式提交。

2.提交作品需附核心设计方案、配上设计理念及解释说明。简单诠释形象创意的含义和创作灵感、吉祥物名字创意说明（衍生品设计）等。 说明文字应提供WORD格式电子文档；

3.材料提交时请将这两部分材料统一放入文件夹中进行压缩，文件夹须命名为：CUVA吉祥物形象设计＋作者姓名＋吉祥物姓名， 以附件形式上传至SuperVolley的投稿邮箱：media@huayugroup360.com 邮件主题注明为吉祥物征集作品，正文要求写清姓名、联系方式（邮箱和电话）， 截止时间为2018年7月29日晚23点59分。

奖项设置：

中标奖（一名）：奖金人民币2万元＋获奖证书＋CUVA纪念品；

最佳创意奖（一名）：奖金人民币5000元＋获奖证书＋CUVA纪念品；

优秀设计奖（两名）：奖金人民币500元＋获奖证书＋CUVA纪念品。

注：奖品中奖金部分为税前金额。
